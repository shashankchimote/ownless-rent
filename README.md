# ownless-rent
RentMate is a college-focused web platform that allows students to list, rent, and sell items within their campus. The goal is to make sharing resources easy, affordable, and trustworthy among students.  This project is an MVP (Minimum Viable Product) built to test the idea inside a college ecosystem before expanding to multiple campuses and cities
